HTML INTERVIEW QUESTIONS


1. Difference between div and span?

Ans: a. div(DIVISION TAG): It is a block level element. It takes whole width of the browser. Used to create segments or division of a webpage.

b. span(SPAN TAG): It is an inline element. It only takes spaces as per the elements present inside of it. Used to create small segments like images, tags, text etc...

SIMILARITIES: Both tags are used for wrapping or containing other HTML elements.

3. What are self closing tags?

Ans:Self closing tags are tags that do not need any closing tags.

Ex: Image tag, Line break tag, Input tag etc...

5. Difference between Block and Inline?

Ans: a. Block: Block elements take whole width of the webpage. It always start on a new line. Ex:div tag,p tag,h1 to h6(headings tags), table tag, lists tag.

b. InlineInline elements take width neccessary for the content inside it. It does not start on a new line. Ex:span tag, image tag, anchor tag, input tag, label tag.

6. Difference between em and strong?

Ans: a.em:The em element is for words that have a stressed emphasis compared to surrounding text, which is often limited to a words or word of a sentence an affects the meaning of the sentence itself. The em makes the content in italic.

b.strong:The strong element is for content that is of greater importance, while the b element is used to draw attention to text without indicating that its more important.

Similarities:Both are inline elements that are used to emphasizing parts of a sentence.
